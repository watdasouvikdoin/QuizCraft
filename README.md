# QuizCraft

QuizCraft is an AI-powered web application that automatically generates multiple-choice questions (MCQs) from academic PDFs and study material. The goal of the project is to make self-assessment and quiz creation faster, easier, and more accessible for both students and educators.

Instead of manually creating questions from notes or textbooks, users can simply upload a PDF and let the system analyze the content using Natural Language Processing (NLP) and Machine Learning techniques to generate quizzes instantly.

## Features

- Upload academic PDFs or study material
- Automatically generate MCQs from content
- AI/NLP-based text processing
- Clean and simple user interface
- Fast quiz generation for quick revision
- Helps students practice and test understanding

## How It Works

1. User uploads a PDF document
2. The backend extracts and processes the text
3. NLP techniques identify important concepts and keywords
4. The system generates MCQs automatically
5. Users can attempt the quiz for self-evaluation

## Motivation

I built QuizCraft to simplify the process of creating practice quizzes from study material. Students often spend a lot of time preparing questions manually while revising. This project was created as an attempt to automate that process and make learning more interactive and efficient.
